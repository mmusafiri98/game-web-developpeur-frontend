Instruction pour le jeu de Tetris

Voici quelques instructions pour jouer au jeu de Tetris:

  • Utilisez les flèches gauche et droite pour déplacer les pièces.
  • Appuyez sur la flèche du haut pour faire pivoter les pièces.
  • Appuyez sur la flèche du bas pour les faire descendre plus rapidement.
  • Appuyez sur "Esc" qui est sur votre clavier pour mettre en pause le jeu.
  • Pour déplacer la pièce, utilisez les touches affichées sur l'image ci-dessus.
